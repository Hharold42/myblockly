Проект на JS с использование библиотеки React.

Более простая версия Blockly Code от Google

для запуска локального сервера и тестирования - README.md

Главный файл находится в public/index.html

в него загружаются изменения с помощью AJAX(неважно что это я и сам не знаю)

для загрузки данных на страницу без обновления страницы используется ReactDOM.createRoot(document.getElementById('root')) - чтобы подключится к оригинальной странице и render(<App />) для отображения обновлений

использованные библиотеки - react-dnd, tailwind, react-dnd-html5-backend
